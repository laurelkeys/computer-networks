### 1. Start a remote object registry:
- Run  `rmiregistry [PORT_NUMBER] &`,  where PORT_NUMBER is the choosen port (1099 by default).

### 2. Run the server:
- Go to the folder **project3.rmi\src\main\java**
- Run `java Server`

### 3. Run the client:
- Go to the folder **project3.rmi\src\main\java**
- Run `java Client`
